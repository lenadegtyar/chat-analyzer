# Chat Analyzer — практика G-Conf

Скрипт анализирует активность участника в чате интенсива и выводит личный дашборд прогресса.

## Задача

Скрипт не работает. Твоя задача — починить его с помощью агента.

**Правила:**
- Не трогай код руками
- Используй агента: опиши проблему, запусти результат, сообщи что получилось
- Повторяй до победы

## Запуск

1. Убедись что установлен Python 3
2. Открой `analyze.py` и поменяй `YOUR_NAME` на своё имя как оно написано в чате
3. Запусти:

```
python analyze.py
```

```
────────────────────────────────────────

────────────────────────────────────────
  Сообщений написано:       8
  Самый активный день:      Saturday

  Вопросов задано:          2
  Streak (дней подряд):     5 🔥
```
